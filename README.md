# Plant-Disease-Detection

### Overview
Plant Disease Detection is a machine learning-based application designed to identify diseases in plants from images. This tool helps farmers and gardeners quickly diagnose plant health issues, enabling timely treatment and better crop management.

### Features
- Detects multiple types of plant diseases from leaf images.
- Provides disease name and severity level.
- User-friendly interface for easy image upload and result display.
- Supports a variety of plants and diseases .
- Suggest the respected fertilizers or pesticides and to autolink to their respective online         buying websites.
### Technologies Used

- TensorFlow / PyTorch
- OpenCV
- Flask /
- Other libraries: NumPy, Pandas, Matplotlib, Efficientnetb4
